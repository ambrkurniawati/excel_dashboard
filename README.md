# 📊 Interactive Sales Performance Dashboard (Excel)

## 🖥️ Dashboard Preview
![Sales Performance Dashboard](Screenshot_Summary Sales 2018.jpeg) 

## 🚀 Key Features & Functionalities
* **Sales Trend Tracking**: Visualizes seasonal revenue fluctuations month-over-month.
* **Shipping Category Analysis**: Combined dual-axis chart evaluating revenue against delivery speed metrics.
* **Geographical Heat Map**: Interactive US map visualizing sales density across states.
* **Interactive Slicers**: Seamless monthly dynamic filtering across all connected charts.

## 🛠️ Tools Used
* **Microsoft Excel**: Data modeling and Pivot Tables.
* **Power Pivot**: Managing underlying relational data.
* **Dynamic Visualization**: Custom combination charts and map charts.

## 📂 How to View the Project
1. Download the `Superstore Dataset.xlsx` file from this repository.
2. Open it using **Microsoft Excel** on desktop to test the interactive slicers.
